  • 생성

2
3

  • 연산

key 값에는 변하는 값을 쓸 수 없음

ex) list 사용불가, tuple은 사용가능

1
2
3

  • 추가

# dic = {1:'a', 2:'b'}
  • 제거

# dic = {1:'a'}
  • 키 사용하기
1

dic = {1:'a', 2:'b'}
dic[1]
dic.get(1)
# 'a'
dic = {'a':'apple', 'b':'banana'}
dic['a']
dic.get('a')
# 'apple'

[] 를 사용하면 존재하지 않는 키를 가져올 경우 오류 발생,
get() 을 사용하면 None 반환
key 값이 없는 경우 default 값을 반환하게 할 수 있다.
a.get(key, default) 와 같이 사용하면 된다.

dic = {1:'a', 2:'b', 3:'c', 4:'d'}
dic.key()
# key List
# [1,2,3,4]
dic.values()
# value List
# ['a','b','c','d']
dic.items()
# key, value List
# [(1, 'a'), (2, 'b'), (3, 'c'), (4, 'd')]

세 가지 모두 수행 결과로 리스트를 얻지만, 기본 리스트의 append, insert, pop, remove, sort 함수는 수행할 수 없음
이 경우 list(dic.key()) 를 이용해 리스트로 변환하여 사용하면 됨

s1 = set([1,2,3,4])
s2 = set([3,4,5,6])

# 교집합
s1 & s2
s1.intersection(s2)
s2.intersection(s1)
# {3,4}

# 합집합
s1 | s2
s1.union(s2)
s2.union(s1)
# {1,2,3,4,5,6}

# 차집합
s1 - s2
s1.difference(s2)
# {1,2}
s2 - s1
s2.difference(s1)
# {5,6}
