# **코드**

- C++ 코드

```cpp
#include <string>

using namespace std;

string solution(int num) {
    string answer = "";
    if(num%2 == 0){
        answer = "Even";
    }
    else{
        answer = "Odd";
    }
    return answer;
}
```

- 파이썬 코드   

```py
def solution(num):
    if num % 2 ==0:
        answer = 'Even'
    else:
        answer = 'Odd'
    return answer
```
