# **문제**

# **접근**  

1번 노드에서부터 각 노드까지의 거리를 구하면 된다.  
따로 가중치가 없으므로 단순하게 BFS를 돌며 몇 번만에 도달했는지만 확인하면 된다.  
<br/>

# **코드**


- 파이썬 코드   

```py
def solution(n, edge):
    answer = 0
    dis = [-1] * (n+1)
    e = {}
    for i in range(1,n+1):
        e[i] = set()
    for a, b in edge:
        e[a].add(b)
        e[b].add(a)
    stack = [1]
    dis[1] = 0
    while stack:
        node = stack.pop(0)
        for item in e[node]:
            if dis[item] == -1:
                dis[item] = dis[node]+1
                stack.append(item)
    max_dis = max(dis)
    for i in range(1,n+1):
        if max_dis == dis[i]:
            answer+=1
    return answer
```
