# **문제**


[약수의 합](https://programmers.co.kr/learn/courses/30/lessons/12928)

<br/>

# **접근**  

n까지 돌며 나누어 떨어지는 수들을 더하면 된다.  
n의 범위에 0도 포함되어 있으므로 주의해야 한다.   
for문의 범위를 1부터 시작해주면 신경쓰지 않아도 된다.  

<br/>

# **코드**


- 파이썬 코드   

```py
def solution(n):
    answer = 0
    for i in range(1,n+1):
        if n%i == 0:
            answer+=i
    return answer
```


- C++ 코드

```cpp
#include <vector>

using namespace std;

int solution(int n) {
    int answer = 0;
    for(int i = 1; i<=n;i++){
        if(n%i == 0){
            answer += i;
        }
    }
    return answer;
}
```
