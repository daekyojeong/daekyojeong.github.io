# 강의


## Python Library   

- Heap   

파이썬에서 라이브러리로 최소 힙(Min Heap) 자료구조를 제공한다.

일반 리스트를 힙처럼 사용할 수 있도록 도와준다.   

C++ 이나 자바에서 처럼 별개의 자료구조를 제공하는게 아니다.   

선언은 일반 리스트와 똑같다.

```
import heapq

heap = []
```

일반 리스트를 변환   

```
heapq.heapify(heap)
```

힙에 원소 추가 및 삭제   

```
heapq.heappush(heap, item)

item = heapq.heappop(heap)
```

삭제하지 않고 최소값 읽기   
**0이외에는 순서가 보장되지 않음**      

```
heap[0]
```

자료구조 자체는 리스트이므로 그 외의 필요한 연산은 리스트 연산과 동일하게 해도 문제 없을 것 같다.
