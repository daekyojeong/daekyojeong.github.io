# **코드**

- C++ 코드

```cpp
#include <string>
#include <vector>

using namespace std;

string solution(vector<string> seoul) {
    string answer = "김서방은 ";
    for(int i = 0; i<seoul.size(); i++){
        if(seoul[i] == "Kim"){
            answer += to_string(i);
            break;
        }
    }
    answer += "에 있다";
    return answer;
}
```

- 파이썬 코드   

```py
def solution(seoul):
    answer = '김서방은 '
    for i, item in enumerate(seoul):
        if item == 'Kim':
            answer += str(i)
            break
    answer += '에 있다'
    return answer
```
