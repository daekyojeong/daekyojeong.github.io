# **코드**

- C++ 코드

```cpp
using namespace std;

bool solution(int x) {
    bool answer = true;
    int num = x, n = 0;
    while(num > 0){
        n += num%10;
        num /= 10;
    }
    if(x%n == 0){
        answer = true;
    }
    else{
        answer = false;
    }
    return answer;
}
```
