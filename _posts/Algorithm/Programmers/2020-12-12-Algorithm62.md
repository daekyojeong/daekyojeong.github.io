# **문제**


[제일 작은 수 제거하기](https://programmers.co.kr/learn/courses/30/lessons/12935)

<br/>

# **접근**  

최소값을 찾고 해당 값을 배열에서 제거하면 된다.  
<br/>

# **코드**


- 파이썬 코드   

```py
def solution(arr):
    minIdx = 0
    for i in range(1,len(arr)):
        if arr[minIdx] > arr[i]:
            minIdx = i
    del arr[minIdx]
    return arr
```


- C++ 코드

```cpp
#include <vector>

using namespace std;

vector<int> solution(vector<int> arr) {
    vector<int> answer;
    int minIdx = 0;
    for(int i = 1; i<arr.size(); i++){
        if(arr[minIdx] > arr[i]){
            minIdx = i;
        }
    }
    arr.erase(arr.begin()+minIdx);
    if(arr.size() == 0){
        arr.push_back(-1);
    }
    return arr;
}
```
