# **문제**

[1085 - 직사각형에서 탈출](https://www.acmicpc.net/problem/1085)

<br/>

# **접근**

상하좌우 가장 가까운 곳을 찾으면 된다.  


<br/>

# **코드**

- 파이썬 코드   

```py
x, y, w, h = map(int, input().split(' '))

print(min(x, w-x, y, h-y))
```
