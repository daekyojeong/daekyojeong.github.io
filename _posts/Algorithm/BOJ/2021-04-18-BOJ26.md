# **문제**

[1284 - 집 주소](https://www.acmicpc.net/problem/1284)

<br/>

# **접근**

1, 0 인 경우만 따로 처리해주면 된다.  



<br/>

# **코드**

- 파이썬 코드   

```py
while True:
    num = list(map(int, list(input())))
    if len(num) == 1 and num[0] == 0:
        break
    ans = 1 + len(num)
    for item in num:
        if item == 1:
            ans += 2
        elif item == 0:
            ans += 4
        else:
            ans += 3
    print(ans)
```
