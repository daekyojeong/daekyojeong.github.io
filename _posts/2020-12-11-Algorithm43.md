# **접근**  

문자열의 길이가 짝수인 경우와 홀수인 경우를 나누어서, 다른 인덱스를 반환    
<br/>

# **코드**

- C++ 코드

```cpp
#include <string>

using namespace std;

string solution(string s) {
    string answer = "";

    int l = s.size() / 2;

    if(s.size() % 2){
        answer = s[l];
    }
    else{
        answer = s.substr(l-1, 2);
    }
    return answer;
}
```

- 파이썬 코드   

```py
def solution(s):
    l = len(s) // 2
    if len(s) % 2:
        answer = s[l]
    else:
        answer = s[l-1:l+1]
    return answer
```
