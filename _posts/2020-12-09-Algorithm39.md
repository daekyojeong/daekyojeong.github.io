# **문제**


[문자열 내 p와 y의 개수](https://programmers.co.kr/learn/courses/30/lessons/12916)

<br/>

# **접근**  
그냥 p, y 개수를 센다.     
<br/>

# **코드**

- C++ 코드

```cpp
#include <string>
#include <iostream>
using namespace std;

bool solution(string s)
{
    int pCnt = 0, yCnt = 0;
    for(int i = 0; i< s.size(); i++){
        if(s[i] == 'p' || s[i] == 'P')
            pCnt++;
        if(s[i] == 'y' || s[i] == 'Y')
            yCnt++;
    }
    if(pCnt == yCnt)
        return true;
    else
        return false;
}
```

- 파이썬 코드
