# **접근**  

문자열 처리만 잘해주면 된다.  

<br/>

# **코드**


- 파이썬 코드   

```py
def solution(s):
    s_list = list(s.split(" "))
    l = []
    for item in s_list:
        if item[0] == '-':
            tmp = -1 * int(item[1:])
        else:
            tmp = int(item[:])
        l.append(tmp)
    l.sort()
    answer = str(l[0]) + " " + str(l[-1])
    return answer
```
