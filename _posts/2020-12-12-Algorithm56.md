<br/>

# **접근**  

10으로 나누며 나머지를 한자리씩 담아주면 된다.   

<br/>

# **코드**


- 파이썬 코드   

```py
def solution(n):
    answer = []
    while n>0:
        answer.append(n%10)
        n //= 10
    return answer
```


- C++ 코드

```cpp
#include <vector>

using namespace std;

vector<int> solution(long long n) {
    vector<int> answer;
    while(n>0){
        answer.push_back(n%10);
        n /= 10;
    }
    return answer;
}
```
