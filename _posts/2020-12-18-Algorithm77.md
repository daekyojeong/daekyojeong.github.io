# **접근**  

A, B를 2로 나눈 몫이 같아질 때 만나게 된다.  
인덱스 조정을 위해 시작을 0부터로 해야 한다.  



<br/>

# **코드**


- 파이썬 코드   

```py
def solution(n,a,b):
    answer = 0
    a -= 1
    b -= 1
    while a!=b:
        a//=2
        b//=2
        answer+=1
    return answer
```


- C++ 코드

```cpp
#include <iostream>

using namespace std;

int solution(int n, int a, int b)
{
    int answer = 0;
    a--;
    b--;
    while (a!=b){
        answer++;
        a/=2;
        b/=2;
    }
    return answer;
}
```
